# CodeAlpha Login & Registration System

## Description
A Login and Registration System developed using C++ and File Handling.

## Features
- User Registration
- Login Authentication
- Duplicate Username Detection
- File Storage
- Menu Driven Program

## Technologies Used
- C++
- File Handling
- Functions

## How To Run

Compile:
```bash
g++ login_system.cpp -o login
```

Run:
```bash
./login
```

## Sample Output
Register → Login → Authentication Success
